# TalentFlow AI Frontend

Enterprise recruitment dashboard prototype built with Next.js 15, TypeScript,
Tailwind CSS, Lucide, and Recharts.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Validation

```bash
npm run lint
npm run build
```

All application data and interactions are local UI fixtures. No backend,
authentication, Gmail, database, or AI API is connected.
